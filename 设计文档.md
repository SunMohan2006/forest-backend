# 林业资源信息管理后台 — 项目设计文档

> 开发周期：2026.7 - 2026.8 | 独立个人项目 | FastAPI + MySQL + SQLAlchemy

---

## 一、功能模块

| 模块 | 功能点 |
|------|--------|
| 用户模块 | 注册、登录、JWT 令牌校验 |
| 林地管理 | 新增林地、删除林地、修改林地、分页查询林地列表、按条件搜索 |
| 图片上传 | 上传遥感图片（关联林地）、查看林地下的图片列表 |
| 数据统计 | 林地总数、按类型分布统计、近一个月新增趋势 |

---

## 二、数据库表设计（4 张表）

### 1. user 用户表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) UNIQUE NOT NULL | 用户名 |
| password | VARCHAR(255) NOT NULL | 密码（BCrypt 加密） |
| role | VARCHAR(20) DEFAULT 'USER' | 角色：ADMIN / USER |
| created_at | DATETIME DEFAULT CURRENT_TIMESTAMP | 注册时间 |

### 2. forest_land 林地表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK AUTO_INCREMENT | 林地ID |
| name | VARCHAR(100) NOT NULL | 林地名称 |
| area | DECIMAL(10,2) | 面积（亩） |
| location | VARCHAR(255) | 地理位置 |
| land_type | VARCHAR(50) | 类型：用材林/防护林/经济林/薪炭林/特用林 |
| description | TEXT | 描述 |
| status | VARCHAR(20) DEFAULT 'ACTIVE' | 状态：ACTIVE/INACTIVE |
| created_by | BIGINT FK → user.id | 创建人 |
| created_at | DATETIME DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 3. forest_image 遥感图片表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK AUTO_INCREMENT | 图片ID |
| land_id | BIGINT FK → forest_land.id | 所属林地 |
| image_url | VARCHAR(500) NOT NULL | 图片存储路径 |
| original_name | VARCHAR(255) | 原始文件名 |
| file_size | BIGINT | 文件大小（字节） |
| uploaded_at | DATETIME DEFAULT CURRENT_TIMESTAMP | 上传时间 |

### 4. operation_log 操作日志表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK AUTO_INCREMENT | 日志ID |
| user_id | BIGINT FK → user.id | 操作人 |
| action | VARCHAR(50) | 操作类型：CREATE/UPDATE/DELETE |
| target | VARCHAR(100) | 操作对象：forest_land/forest_image |
| target_id | BIGINT | 操作对象ID |
| detail | VARCHAR(500) | 操作详情 |
| created_at | DATETIME DEFAULT CURRENT_TIMESTAMP | 操作时间 |

### ER 关系

```
user (1) ──────< (N) forest_land (1) ──────< (N) forest_image
  │
  └───────────< (N) operation_log
```

## 三、API 接口清单

### 用户模块

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | /api/user/register | 用户注册 | 否 |
| POST | /api/user/login | 用户登录，返回 JWT | 否 |

### 林地管理

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | /api/forest-land | 新增林地 | 是 |
| DELETE | /api/forest-land/{id} | 删除林地 | 是 |
| PUT | /api/forest-land/{id} | 修改林地信息 | 是 |
| GET | /api/forest-land/{id} | 查询单个林地详情 | 是 |
| GET | /api/forest-land/page | 分页查询林地列表 | 是 |
| GET | /api/forest-land/search | 按名称/类型/位置搜索 | 是 |

### 图片管理

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | /api/forest-image/upload | 上传遥感图片 | 是 |
| GET | /api/forest-image/land/{landId} | 查看某林地的图片列表 | 是 |

### 数据统计

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| GET | /api/statistics/overview | 总览：林地总数、图片总数 | 是 |
| GET | /api/statistics/by-type | 按林地类型分布统计 | 是 |
| GET | /api/statistics/monthly-trend | 近30天新增林地趋势 | 是 |

## 四、技术架构

```
┌──────────────────────────────────────────┐
│         前端（极简 HTML / Vue）            │
├──────────────────────────────────────────┤
│       FastAPI 自动生成 Swagger 文档        │
│     (面试官浏览器打开 /docs 直接调接口)      │
├──────────────────────────────────────────┤
│           Router 路由层                   │
│   user / forest_land / forest_image       │
│            / statistics                   │
├──────────────────────────────────────────┤
│          Service 业务逻辑层                │
├──────────────────────────────────────────┤
│       SQLAlchemy ORM 数据访问层            │
├──────────────────────────────────────────┤
│          MySQL 数据库                      │
├──────────────────────────────────────────┤
│     文件存储：本地 uploads/ 目录            │
└──────────────────────────────────────────┘
```
